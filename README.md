## Fonctionnalites ajoutees
- Recherche multi-criteres par nom, type, ID avec compteur de resultats
- Filtre par type et tri (nom, ID, HP, attaque, defense)
- Pagination 20 par 20 avec pages numerotees
- Upload d'image (URL ou fichier) avec previsualisation
- Validations front (champs requis, formats, erreurs detaillees)
- Validations serveur (statuts HTTP + message clair)
- Cartes enrichies (tilt leger, ratios fixes, badges, barres de stats)
- Header anime + gradients dynamiques selon type (page details)
- Radar chart des stats sur la page details (mise à jour en live si les statistiques sont modifiées)
- Mode sombre (toggle + persistance)
- UI responsive (grilles 3→2→1 colonnes, header adaptatif)
- Modification du fond de l'ajout de la carte en fonction des types sélectionnés
- Bouton pour remonter en haut de la page rapidement
